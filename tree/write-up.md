# Daily Reflection Tree – Design Write-up

## Approach
I designed this system as a structured reflection conversation rather than a survey. The goal was to guide users through three psychological axes progressively.

## Axes Covered
1. Locus of Control
2. Contribution vs Entitlement
3. Radius of Concern

## Psychological Basis
- Rotter (Locus of Control)
- Carol Dweck (Growth Mindset)
- Organizational Citizenship Behavior
- Maslow (Self-Transcendence)

## Design Decisions

### Deterministic System
- No AI at runtime
- Fixed options only
- Predefined branching

### Realistic Options
Options reflect real workplace behavior rather than ideal answers.

### Non-judgmental Reflection
The tone encourages awareness, not blame.

### Progressive Flow
Control → Contribution → Perspective

## AI Usage
AI was used for idea generation and testing, but all final decisions and structure were manually designed.

## Improvements
- More branching depth
- Role-specific trees
- UI implementation

## Conclusion
This project demonstrates how human psychology can be encoded into a deterministic system for structured reflection.
